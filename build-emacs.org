I use the following commands to build GNU Emacs from source.

#+begin_src shell
  git clone https://git.savannah.gnu.org/git/emacs.git
  cd emacs
  ./autogen.sh
  ./configure --with-pgtk --with-tree-sitter --prefix=$HOME/opt/emacs  # ./configure --help
  make
  make install
#+end_src

For more details, see [[https://git.savannah.gnu.org/cgit/emacs.git/plain/INSTALL.REPO][INSTALL.REPO]].
